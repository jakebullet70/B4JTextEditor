# Editor Keyboard Shortcuts

The editing surface is a **RichTextFX CodeArea**. Below are its built-in key
bindings, plus this app's menu shortcuts. (On macOS use **Cmd** where this
lists **Ctrl**.)

## Caret movement

- `Left` / `Right` - move one character
- `Ctrl+Left` / `Ctrl+Right` - move one word
- `Up` / `Down` - move one line
- `Home` / `End` - start / end of line
- `Ctrl+Home` / `Ctrl+End` - start / end of document
- `Page Up` / `Page Down` - move one page

## Selection

- `Shift+<any movement>` - extend the selection in that direction
- `Ctrl+A` - select all
- Double-click - select the word
- Triple-click - select the paragraph
- Click then `Shift+Click` - select a range

## Editing

- `Tab` - insert spaces (or a tab) - see Setup
- `Enter` - new line (keeps the current line's indent - see Setup)
- `Backspace` - delete the character before the caret
- `Delete` - delete the character after the caret
- `Ctrl+Backspace` - delete the word before the caret
- `Ctrl+Delete` - delete the word after the caret

## Clipboard

- `Ctrl+C` / `Ctrl+Insert` - copy
- `Ctrl+X` / `Shift+Delete` - cut
- `Ctrl+V` / `Shift+Insert` - paste

## Undo / redo

- `Ctrl+Z` - undo
- `Ctrl+Y` / `Ctrl+Shift+Z` - redo

## Application shortcuts

- `Ctrl+N` - new tab
- `Ctrl+O` - open file
- `Ctrl+S` - save
- `Ctrl+Shift+S` - save as
- `Ctrl+W` - close tab
- `Ctrl+Q` - exit
- `Ctrl+F` - find
- `F3` - find next
- `Ctrl+R` - replace
- `Ctrl+G` - go to line
- `Alt+Z` - toggle word wrap
- `Ctrl+Page Up` / `Ctrl+Page Down` - previous / next tab
